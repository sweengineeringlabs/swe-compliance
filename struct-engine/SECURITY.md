# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue at:

https://github.com/sweengineeringlabs/swe-compliance/issues

## Scope

struct-engine performs read-only local file system scans. It does not:

- Access the network
- Execute discovered code
- Modify scanned projects
