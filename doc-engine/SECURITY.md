# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in doc-engine, please report it responsibly.

1. Do not open a public issue
2. Email the maintainers with a description of the vulnerability
3. Include steps to reproduce, if possible
4. Allow reasonable time for a fix before public disclosure

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days.
